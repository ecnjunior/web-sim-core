# 🎮 webgame-sim-core

A lightweight, zero-dependency TypeScript engine designed for state management, deterministic logic, and tick-based updates in browser-based games and simulation applications.

## 🚀 Features

- **Deterministic PRNG:** Seeded pseudo-random number generation for reproducible game states and simulations.
- **Tick Engine:** Lightweight timing controller for turn-based or tick-driven game loops.
- **Zero Dependencies:** Pure TypeScript logic built for maximum performance and portability.

## 📦 Installation

```bash
npm install webgame-sim-core
